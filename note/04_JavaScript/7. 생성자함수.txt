7. 생성자함수

-생성자함수
function 생성자함수명(속성값들) {
            ...
      }
var 객체명 = new 생성자함수명(속성값들); 

--프로토타입(prototype) : 생성자 함수로 생성된 객체가 공통으로 가지는 공간

--new 키워드(생성자 함수로 만든 객체 : 객체 또는 인스턴스)
키워드로 함수 호출시 객체를 위한 공간 생성(this 키워드가 해당 공간을 의미)
>이외의 경우 일반적으로 this 키워드는 window객체를 의미함
해당 객체가 어떠한 생성자 함수로 생성됐는지 확인할 때 instanceof 키워드 사용 가능.

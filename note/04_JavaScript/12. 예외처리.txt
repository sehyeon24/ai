12. 예외처리
프로그램이 실행되는 동안 문제가 발생하면 프로그램이 자동으로 중단되므로 이때 프로그램이 실행되는 동안 문제가 발생하면 프로그램이 자동으로 중단되도록 하는 것

-예외발생 
①잘못된 코드 작성 ②사용자가 개발자가 원하지 않는 방향으로 프로그램을 사용했을 때

ex).
<script>
 try{
            var array = new Array(9999999999);
            // 배열의 크기는 최대 4,294,967,295(2^32 - 1)개까지 가능
        } catch (exception) {
            alert('예외 발생하였습니다');
            alert(exception);
        } finally {
            alert('예외 발생 가능 부분을 통과했습니다');
        }
    </script>

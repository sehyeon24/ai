-공간분할태그(레이아웃태그)
1) div(블록형식 태그) : block 형식으로 공간을 분할 . 하나의 블록형식의 태그므로 개행포함 ex)p, ul안의 li


2)span(라인형식 태그) : inline 형식으로 공간을 분할

-시맨틱태그 구
 header : 해더
 nav : 내비게이션
 aside : 사이드에 위치하는 공간
 section : 여러 중심 내용을 감싸는 공간
 article : 글자가 많이 들어가는 부분
 footer : 꼬리말

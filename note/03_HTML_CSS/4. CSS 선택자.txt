4. CSS 선택자

1) 태그선택자 : 특정 태그를 선택하여 CSS속성을 적용
2) 전체(*) 선택자 ; 페이지 초기화 : 전체라는 의미의 * 선택자를 사용하면 문서 전체를 선택
3) id(#)선택자 class(.)선택자 : 태그의 id와 clas를 이용하여 CSS에 속성을 설정
4) 혼합선택자 : tag, id, class를 혼합하여 CSS에서 속성을 설정
5) 속성 선택자 : tag의 속성을 선택하여 CSS의 속성을 설정
6) 후손선택자, 자손선택자(바로아랫단) : tag의 속성을 선택하여 CSS의 속성을 설정
7) 동위(형제)선택자(+하나, ~전체) : 태그의 동등한 위치를 판단하여 CSS속성을 설정
8) 반응선택자(ex.마우스를 올릴때 반응) : 마우스를 올려 놓았을 때(hover상태)의 반응에 따른 속성을 설정
9) 상태선택자(ex. input에 focus된 상태) : 상태에 따라 CSS속성이 변화하는 설정 ex). input태그의 경우 focus가 맞춰져 있을 때
10) 구조선택자 : 구조에 따라 CSS 속성이 변화하는 설정
11) 문자선택자 : 특정 문자만 선택하여 CSS 속성을 설정
12) 링크선택자 : 문서에 링크(href) 되어 있는 문자를 선택하여, CSS 속성을 설정

2. HTML-I (글자관련 태그, 목록 태그, 표 태그, 이미지 태그, 오디오 태그)

-글자관련 태그
h1(제목), p(본문), br(개행), a(앵커), b, I, sup, del 태그

-목록태그
1) ul : 순서가 없는 목록태그	2) ol : 순서가 있는 목록 태그

-표태그
1) tr : 표 내부의 행 태그
2) th : 행 내부의 열. 제목 셀 태그
3) td : 행 내부의 열. 일반 셀 태그
4) thead : 표의 제목부분 (선택)
5) tbody : 표의 본문 (선택)
6) tfoot : 표의 하단 (선택)

-이미지태그
1) src : 이미지의 경로 지정
2) alt : 이미지가 없을 때 나오는 글자 지정
3) width : 이미지의 너비 지정 (css 부분에 설정해주는 경우가 더 많다)
4) height : 이미지의 높이 지정 (css 부분에 설정해주는 경우가 더 많다)

-오디오태그
1) src : 오디오 파일의 경로 지정
2) preload : 오디오를 재생하기 전에 모두 불러올지 지정
3) autoplay : 오디오를 자동 재생할지 지정 (인터넷 익스플로어에서 작동)
4) loop : 오디오를 반복할지 지정
5) conrols : 오디오 재생 도구를 출력할지 지정

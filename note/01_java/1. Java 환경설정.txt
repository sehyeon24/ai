1. Java 언어의 특징
- Virtual Machin 운영체제(윈도우, 리눅스, 맥OS)의 영향을 받지 않는다.
- Garbage Collection 개발자는 메모리 관리는 하지 않는다. (JVM이 메모리 자동 관리)
- Object Oriented Program(OOP) 객체지향 언어로 개발과 유지보수를 효율적으로 진행할 수 있다
- 다중 Thread 를 지원한다.
- 기능 지원 >> 접근에 대한 권한을 변수로 처리 >> Access Modifier(접근제한자)

2. Java 프로그래밍 환경 설정과 실행 방법
- 운영체제에 맞는 JDK 설치
- Java 환경 변수 설정
- 설치한 JDK 버전에 맞는 이클립스 다운로드

1. 파이썬 개요 및 개발환경 구성
-주피터 노트북 사용법(큰제목)
shift + Enter : 포커스된 셀이 실행되고, 커서가 다음셀로 이동
alt + Enter : 포커스된 셀이 실행되고 다음셀이 생성된 후 다음셀로 이동
ctrl + Enter : 포커스된 셀이 실행되고 포커스 이동이 없음
# 셀 하나하나에 파이썬 실행 소스 입력 -> 실행

-ESC키 + (핫키)
dd : 셀삭제
a : 윗셀 추가
b : 아랫셀 추가
m : 마크다운셀(문서셀)로 변경
y : 코드셀(소스셀)로 변경
h : 핫키 안내
o : 실행결과 접기

-마크다운이란?
코드와 함께 문서화를 할 수 있음.
다양한 문서화 문법이 있음.

9. 파일 i/o 프로그래밍
- 파일에 데이터를 저장하고 불러오기
w 또는 wt : (write)  기존의 파일이 있는 경우 덮어쓰기, 파일이 없는 경우 생성하여 쓰기
- 쓰기 모드는 폴더가 없는 경우 예외 발생
a 또는 at : (append) 기존의 파일이 있는 경우 (맨 뒤에) append, 파일이 없는 경우 생성하여 쓰기
r 또는 rt : (read)   기존의 파일을 읽는 경우, 파일이 없는 경우 예외 발생
-- euc-kr : 한글 완성형 (믜X)
-- cp949 : 확장된 한글 완성형 (믜O) - 엑셀에서
-- utf-8 : 한글 조합형 (초중종성 따로 조합해서 저장) - 주피터노트북에서

- 피클을 이용한 객체 저장 및 불러오기
-- 형식이 있는 텍스트 데이터 불러오기
-- 피클링 : 바이너리를 사용해 저장할 때는 피클을 이용해야 함

- JSON 파일에 저장하고 불러오기(dump, load)
-- JSON dump(파일에 저장)
딕셔너리 리스트 데이터를 json파일로
객체 리스트 데이터를 json파일로
-- JSON load(파일 읽기)
json파일을 딕셔너리 리스트 변수로 읽기
json파일을 객체 리스트 변수로 읽기

- hdf5파일 쓰기/읽기(딕셔너리, 데이터프레임)
-- hdf5파일 쓰기
딕셔너리 데이터를 hdf5파일에 쓰기
데이터프레임 데이터를 hdf5파일에 쓰기
-- hdf5파일 읽기
hdf5파일에서 데이터프레임으로 읽기
hdf5파일에서 딕셔너리로 읽기
